Technical Guidelines

REST API

Each implemented REST services must be documented (see /users directory): REST API DOCUMENTATION

  • How to configure the SCAVA Gateway in order to integrate a new REST service Customers access SCAVA services through the SCAVA API Gateway. This guideline present how to configure the API Gateway to integrate new remote service provider.

  • How to implement Restlet services Guideline which describe how to integrate and implement a REST service in SCAVA platform using the RESTLET framework.
